編輯器的故事

編輯器


  • 一般編輯器
  • 以「模式」為基本概念的編輯器
    • 靈活
    • 自由組合任務
    • 效率高

編輯器的模式



  • normal mode
  • visual mode
  • visual line mode
  • visual block mode
  • insert mode
  • replace mode

修改一段文字


一般編輯器

  • 把「看看今日的熱門」改成「看看本週熱門」
  • 動作分解
    1. 移動游標
    2. 選取文字
    3. 刪除文字
    4. 輸入新文字

修改一段文字


模式編輯器-1

  • 把「看看今日的熱門」改成「看看本週熱門」
  • 動作分解
    1. 移動游標 (normal mode)
    2. 選擇文字 (visual mode)
    3. 刪除文字 (any mode)
    4. 輸入新文字 (insert mode)

修改一段文字


模式編輯器-2

  • 把「看看今日的熱門」改成「看看本週熱門」
  • 動作分解
    1. 移動游標 (normal mode)
    2. 選擇文字 (visual mode)
    3. 取代文字 (replace mode)

一般編輯器－複製、貼上


  • ⌘ + c
  • ⌘ + v



模式編輯器－visual mode


  • y
  • p


非 insert mode，不會輸入！

一般編輯器－複製動作分解


  • 選取文字
  • 按著 ⌘
  • 暫時進入 visual mode
  • 按 c 複製



模式編輯器－visual mode


  • 選取文字
  • 已經在 visual mode
  • y

小結



  • 一般編輯器一直處於 insert mode
  • 一般編輯器得靠特殊鍵進入其他模式（如：ctrl、alt 等）
  • 模式編輯器，設計哲學是將各種功能分解成最小任務
  • 利用快速鍵完成多項小任務
  • 靈活多變，快速達到目的（大任務）
  • ref: http://blog.vgod.tw/2009/12/07/text-editing/





快速鍵工作坊

Mac 特殊鍵

windows 下大部分快速鍵，要將 command 改成 ctrl

複製、貼上、剪下


  • ⌘ + c
  • ⌘ + v
  • ⌘ + x



頁面移動


  • fn + ↑ (page up)
  • fn + ↓ (page down)
  • fn + ← (home)
  • fn + → (end)

文字間移動


  • alt + ←
  • alt + →
  • ctrl + ← (windows)
  • ctrl + → (windows)



游標移動


  • ⌘ + ↑
  • ⌘ + ↓
  • ⌘ + ←
  • ⌘ + →



選取


shift + ↑ ↓ ← →



全選


⌘ + a

範圍選取（游標移動＋選取）


⌘ + shift + ↑ ↓ ← →



文字範圍選取（文字移動＋選取）


⌘ + shift + ← →

快速切換 tab（chrome 為例）



  • ⌘ + 1~8
  • ⌘ + 9
  • ⌘ + alt + ←
  • ⌘ + alt + →

tab 開啟、關閉（chrome 為例）


  • ⌘ + t
  • ⌘ + shift + t



一般視窗開啟、關閉


  • ⌘ + w
  • ⌘ + n
  • ⌘ + q (mac)
  • alt + F4 (windows)
  • special: ⌘ + shift + n (chrome 無痕視窗)

切換程式視窗



  • ⌘ + tab (mac)
  • alt + tab (windows)
  • win + tab (windows 7 only)
  • ⌘ + ` （mac only 相同程式間視窗切換）

上一頁/下一頁

  • ⌘ + [
  • ⌘ + ]



存檔

  • ⌘ + s



搜尋

  • ⌘ + f

擷取畫面



  • ⌘ + shift + 3
  • ⌘ + shift + 4
  • screen (windows)
  • ⌘ + shift + 5 (skitch)

頁面放大縮小



  • ⌘ + +
  • ⌘ + -
  • ⌘ + 0

觸控板



請打開「系統偏好設定」的「觸控式軌跡版」觀看操作影片
